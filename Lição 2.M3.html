Cores e Imagens

As cores são um dos aspectos mais importantes do design de uma página. No CSS, podemos definir as cores de vários elementos, como o fundo (background), o texto, as bordas, entre outros.

Definindo Cores no CSS

Existem várias formas de especificar cores no CSS. Vou explicar as mais comuns:

1. Cores por nome

Você pode usar o nome de cores predefinidas no CSS, como red, blue, green, etc. Veja o exemplo.

body {

background-color: lightblue; (Cor de fundo do body)

color: darkgreen; (Cor do texto)

}

Existem mais de 140 nomes de cores predefinidos que você pode usar, como white, black, gray, yellow, aqua, etc.

2. Cores usando valores Hexadecimais (Hex)

As cores também podem ser definidas usando valores hexadecimais. Um valor hexadecimal começa com # seguido por 6 caracteres (dígitos de 0-9 e letras de A-F) que representam a intensidade de vermelho (R), verde (G) e azul (B) na cor (isso é conhecido como modelo RGB).

Exemplo de cor hexadecimal:

h1 {

color: #ff6347;

}

No exemplo acima, #ff6347 é um tom de vermelho (a intensidade de R, G e B é representada pelos dois primeiros, dois intermediários e os dois últimos dígitos).

3. Cores usando RGB

RGB é o modelo de cores em que as cores são representadas pela mistura de vermelho (Red), verde (Green) e azul (Blue). Veja o exemplo.

p {

color: rgb(255, 99, 71);

}

No modelo RGB, os valores de cada cor variam de 0 a 255, sendo 0 a ausência de cor e 255 a intensidade máxima.

4. Cores usando HSL

HSL (Hue, Saturation, Lightness) é outro modelo de cores. Em HSL, você define a cor em três parâmetros:

• Hue (matiz): Representa a cor em um círculo de 360° (0° = vermelho, 120° = verde, 240° = azul, etc.).

• Saturation (saturação): Representa a intensidade da cor (0% = sem cor, 100% = cor total).

• Lightness (luminosidade): Representa a quantidade de luz na cor (0% = preto, 100% = branco).

footer {

background-color: hsl(120, 60%, 40%); (Cor verde com 60% de saturação e 40% de luminosidade)

}


Gradientes no CSS: Focando no Linear Gradient

Um gradiente é uma transição gradual entre duas ou mais cores. Em CSS, você pode usar gradientes para criar efeitos de cor muito interessantes, como fundos, botões e até textos.

O que é um Linear Gradient?

Um linear gradient é um tipo de gradiente onde as cores se mesclam ao longo de uma linha reta. Ou seja, as cores mudam de forma linear, geralmente de um ponto inicial até um ponto final (que pode ser configurado com diferentes ângulos ou direções).


Sintaxe Básica do Linear Gradient

A propriedade para criar um linear gradient no CSS é background: linear-gradient.

Aqui está a sintaxe básica:

element {

background: linear-gradient(direção, cor1, cor2, cor3, ...);

}

• direção: A direção da linha do gradiente. Pode ser um ângulo (em graus) ou uma direção específica, como to right, to top, to bottom left, etc.

• cor1, cor2, cor3, ...: As cores que formarão o gradiente. Você pode especificar mais de duas cores para criar transições mais complexas.

Exemplo simples de linear gradient:

body {

background: linear-gradient(to right, red, yellow);

}

Neste exemplo, o gradiente vai do vermelho (à esquerda) para o amarelo (à direita). O to right define que a direção do gradiente será da esquerda para a direita.

Você pode usar várias cores em um único gradiente. Por exemplo, criando um gradiente com 3 ou mais cores:

Body {

background: linear-gradient(to right, red, yellow, green);

}

Neste caso, o gradiente vai do vermelho para o amarelo e depois para o verde.

Você também pode especificar onde cada cor começa e termina no gradiente. Isso é feito adicionando um valor percentual ou em unidades de comprimento ao lado da cor.

Body {

background: linear-gradient(to right, red 10%, yellow 50%, green 90%);

}

• red 10% significa que a cor vermelha começará até 10% do caminho do gradiente.

• yellow 50% significa que o amarelo aparecerá no meio (50% do caminho).

• green 90% significa que o verde começará a 90% do caminho.

Isso permite controlar com precisão a posição de cada cor ao longo do gradiente.


Usando Gradientes em Textos

Sim, você também pode aplicar gradientes a textos! Para isso, é necessário usar a propriedade background-clip (que determina como o fundo é cortado) junto com a propriedade linear-gradient:

h1 {

background-color: linear-gradient(to left, red, yellow, green);

}

Aqui, o gradiente será aplicado ao texto do h1, criando um efeito de texto com cores graduais.


Imagens no CSS

Agora vamos falar sobre como trabalhar com imagens no CSS. As imagens podem ser usadas como fundo de elemento ou também como parte do conteúdo visual do site.


Usando Imagens como Fundo (background-image)

Uma das maneiras mais comuns de usar imagens no CSS é aplicá-las como fundo de elementos. A propriedade background-image define a imagem de fundo de um elemento. Veja o exemplo:

Body {

background-image: url(image.jpg);

background-size: cover;

background-position: center;

}

• url(): Dentro dos parênteses, você coloca o caminho para a imagem.

• background-size: cover;: A imagem vai cobrir toda a área do elemento, mas mantendo sua proporção.

• background-position: Define o posicionamento da imagem dentro do elemento. Pode ser left, center, right, top, bottom, ou valores em pixels ou porcentagem.

Você pode alterar a opacidade de imagens e outros elementos usando a propriedade opacity. Um valor de 1 significa totalmente opaco e 0 significa totalmente transparente. Veja o exemplo:

img {

opacity: 0.5; ( A imagem terá 50% de transparência)

}


Transições Simples

Transições (Transitions)

As transições no CSS são usadas para criar efeitos suaves quando uma propriedade de um elemento muda. Com isso, você consegue fazer algo como uma cor de fundo que muda gradualmente quando o usuário passa o mouse sobre o elemento. Em resumo, você "transita" de um estado para outro de forma animada.

Como funciona:

• transition é a propriedade principal que define a duração e o tipo de transição entre os estados.

• Você define a propriedade, o tempo (duração), e opcionalmente, o tipo de curva de aceleração (como ease, linear, etc.).

Sintaxe básica:

elemento {

transition: propriedade duração tipo-de-velocidade;

}


Exemplo simples:

(Estado inicial)

button {

background-color: blue;

color: white;

padding: 10px 20px;

border: none;

cursor: pointer;

transition: background-color 0.3s ease;

}

( Estado quando o mouse passa sobre o botão)

button:hover {

background-color: green;

}

Neste exemplo, quando o usuário passar o mouse sobre o botão, a cor de fundo muda de azul para verde de forma suave em 0.3 segundos.

Vamos aprender a fazer mais uma transição com os seguintes requisitos:

• Quando o usuário passa o mouse sobre o elemento (pode ser um texto ou uma imagem), ele se move para cima.

• A animação deve ser suave, ou seja, um movimento gradual, e não abrupto.

Para isso, vamos usar a propriedade transform, que permite mover, girar, redimensionar, etc., e combinar com a propriedade transition para criar o efeito suave.

Estrutura básica:

• transform: translateY(): Essa propriedade vai mover o elemento no eixo Y (vertical). Um valor negativo faz o elemento subir (mover para cima), e um valor positivo faz o elemento descer.

• transition: Vai permitir que a mudança de posição aconteça de forma suave.


Exemplo 1: Texto subindo ao passar o mouse

Aqui temos um exemplo onde o texto se move para cima quando o mouse passa por cima dele.

HTML 
    
<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Transição de Texto</title>
<style>

.texto {

font-size: 2rem;

font-weight: bold;

color: #333;

display: inline-block;

transition: transform 0.3s ease;

>}

.texto:hover {

transform: translateY(-10px);

}

    
</style>
</head>
<body>
<p class="texto">Passe o mouse sobre mim!</p>
</body>
</html>

O que acontece aqui:

• Quando o mouse não está sobre o texto, ele permanece na posição original.

• Quando o mouse passa sobre o texto, a propriedade transform com translateY(-10px) é aplicada, movendo o texto 10px para cima.

• O transition: transform 0.3s ease cria uma transição suave de 0.3 segundos, o que faz com que o movimento seja gradual, não instantâneo.

• O efeito é acionado quando o mouse entra e sai do elemento (-hover-).

Exemplo 2: Imagem subindo ao passar o mouse

Agora, vamos aplicar o mesmo efeito em uma imagem:

HTML 

<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Transição de Imagem</title>
<style>

.imagem {

width: 300px;

height: 200px;

transition: transform 0.3s ease;

}

.imagem:hover {

transform: translateY(-15px);

}


</style>
</head>
<body>
<img src="img.jpg" width=”300px” alt="Imagem exemplo" class="imagem">
</body>
</html>

Explicação do exemplo da imagem:

• A imagem tem uma largura de 300px.

• Ao passar o mouse sobre a imagem, a propriedade transform: translateY(-15px) é ativada, fazendo a imagem subir 15px.

• A transição é suave porque utilizamos transition: transform 0.3s ease, ou seja, a movimentação de 15px vai acontecer em 0.3 segundos com aceleração e desaceleração suaves.

• Quando o mouse sai da imagem, a transição ocorre no caminho contrário, trazendo a imagem de volta para a posição original.


Curvas de Aceleração

Tanto nas transições quanto nas animações, você pode definir a aceleração usando valores como:

• ease: começa mais devagar, acelera e depois desacelera no final.

• linear: mantém a velocidade constante.

• ease-in: começa devagar e acelera no final.

• ease-out: começa rápido e desacelera no final.

• ease-in-out: começa e termina devagar, com aceleração no meio.
