Criação de Listas, Tabelas e Formulários

 Criação de listas

No HTML, as listas são usadas para organizar e apresentar informações de forma clara e estruturada. Existem três tipos principais de listas em HTML: listas não ordenadas, listas ordenadas e listas de definição. Vamos falar sobre cada uma delas.

1. Listas não ordenadas

A lista não ordenada é usada quando os itens da lista não precisam ser classificados de maneira numerada. A tag <-ul-> define o início e o fim da lista não ordenada.

Dentro de <-ul->, usamos a tag <-li-> para representar cada item da lista. Por exemplo:

    
<ul>
<li>Maçã </li>
<li>Banana </li>
<li>Uva </li>
</ul>

Resultado no navegador:

  • Maçã
  • Banana
  • Uva

2. Listas ordenadas

A lista ordenada é usada quando a ordem dos itens importa, como em etapas de um processo ou uma classificação. Por padrão, os itens em uma lista ordenada são numerados. A tag <-ol-> define o início e o fim da lista ordenada. Dentro de <-ol->, usamos a tag <-li-> para representar cada item da lista. Por exemplo:

    
<ol>
<li>Primeiro passo</li>
<li>Segundo passo</li>
<li>Terceiro passo</li>
</ol>

Resultado no navegador:

  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo

Se você preferir letras maiúsculas ou minúsculas, ou números romanos… para a numeração, use o atributo type=:

     
<ol type="A">
<li>Primeiro passo</li>
<li>Segundo passo</li>
<li>Terceiro passo</li>
</ol>

Resultado no navegador:

  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo

3. Listas de definição

A lista de definição é usada quando você precisa apresentar uma lista de termos e suas definições, como um glossário ou dicionário.

A tag <-dl-> define o início e o fim da lista de definição.

A tag <-dt-> é usada para definir o termo.

A tag <-dd-> é usada para definir a descrição ou definição do termo.

Por exemplo:

    
<dl>
<dt>HTML</dt>
<dd>HyperText Markup Language.</dd>
<dt>CSS</dt>
<dd>Cascading Style Sheets.</dd>
</dl>

Resultado no navegador:

HTML
HyperText Markup Language.
CSS
Cascading Style Sheets.

N.B.: O atributo type não se aplica nas listas de definição.


 Criação de tabelas

A criação de tabelas em HTML permite organizar dados de forma estruturada em linhas e colunas.A estrutura de uma tabela é composta principalmente pelas seguintes tags:

<-table->: Define o início e o fim da tabela.

<-tr-> (Table Row): Define uma linha da tabela.

<-th-> (Table Header): Define uma célula de cabeçalho, geralmente usada para títulos de coluna.

<-td-> (Table Data): Define uma célula de dados na tabela.

Por exemplo: 

    
<table border="1">
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Ana</td>
<td>25</>
</tr>
<tr>
<td>João</td>
<td>30</td>
</tr>
</table>

Resultdo no navegador:

Nome Idade
Ana 25
João 30

No exemplo acima, usamos o atributo border="1" na tag <-table->, o que adiciona uma borda à tabela.

Em vez de usar o atributo border="1", você pode usar CSS para personalizar a aparência das tabelas. Mas esse é um assunto que abordaremos no próximo módulo.

Se não for especificada uma moldura para a tabela, ela irá aparecer sem limites, como mostra acima. Para que a tabela tenha limites, use o atributo bolder-collapse na folha de estilo (CSS) para unir os limites. Ficaria assim:

CSS

Table, th, td {

Border: 1 ph:

Border-collapse: collapse;

}

Outro Elemento útil em Tabela é o <-caption->. Que é usado para adicionar um título à tabela, que fica acima dela.

Por exemplo: 


<table border="1">
<caption<Informações do individuo</caption>
<tr>
<th>Nome</th>
<th>Idade</th>
</tr>
<tr>
<td>Ana</td>
<td>25</>
</tr>
<tr>
<td>João</td>
<td>30</td>
</tr>
</table>

Resultado no navegador:

Informações do individuo
Nome Idade
Ana 25
João 30

Atributo colspan e rowspan

>Os atributos colspan e rowspan são usados em tabelas HTML para mesclar células. Eles permitem que você crie células que se estendem por mais de uma coluna (colspan) ou mais de uma linha (rowspan).

Atributo colspan

O atributo colspan permite que uma célula de uma tabela ocupe várias colunas.

Exemplo de colspan:

Vamos supor que temos uma tabela com 3 colunas, mas queremos que a primeira linha ocupe toda a largura da tabela (3 colunas). Veja o exemplo:

HTML 

<table border="1">
<tr>
<td colspan="3">Cabeçalho que ocupa 3 colunas</td>
</tr>
<tr>
<td>Celula 1</td>
<td>Celula 2</td>
<td>Celula 3</td>
</tr>
</table>

Resultado no navegador:

Cabeçalho que ocupa 3 colunas
Celula 1 Celula 2 Celula 3

Atributo rowspan

O atributo rowspan permite que uma célula de uma tabela ocupe várias linhas. Veja o exemplo:

HTML 
    
<table border="1">
<tr>
<td rowspan="2">Celula que ocupa 2 linhas</td>
<td>Celula 1</td>
<td>Celula 2</td>
</tr>
<tr>
<td>Celula 3</td>
<td>Celula 4</td>
</tr>
</table>

Resultado no navegador:

Celula que ocupa 2 linhas Celula 1 Celula 2
Celula 3 Celula 4

Atributo colspan e rowspan juntos

Veja o exemplo:

    
<table border="1">
<tr>
<td rowspan="2">Celula A (2 linhas)</td>
<td colspan="2">Celula B (2 colunas)</td>
</tr>
<tr>
<td>Celula C</td>
<td>Celula D</td>
</tr>
<tr>
<td>Celula E</td>
<td>Celula F</td>
<td>Celula G</td>
</tr>
</table>

Resultado no nagevador:

Celula A (2 linhas) Celula B (2 colunas)
Celula C Celula D
Celula E Celula F Celula G

 Criação de formulários

A criação de formulários em HTML é uma das partes fundamentais para a interação com os usuários em uma página web. Formulários são usados para coletar dados de usuários, como nome, e-mail, senhas, comentários, etc.

Eis abaixo, a estrutura básica de um formulário:

    
<form action="processar.php" method="POST">

<label for="nome">Nome:</label>
<input type="text" id="nome" name="nome">

<label for="email">E-mail:</label>
<input type="email" id="email" name="email">

<label for="mensagem">Mensagem:</label>
<textarea id="mensagem" name="mensagem"></textarea>

<button type="submit">Enviar</button>
</form>

Explicação dos elementos do formulário:

 <-form->:Esta tag contém todo o formulário. O atributo action define para qual URL os dados do formulário serão enviados, e o atributo method define como esses dados serão enviados:

1. method="GET":Envia os dados pela URL, tornando-os visíveis na barra de endereços.

2. method="POST": Envia os dados de forma oculta no corpo da requisição, sendo mais seguro para dados confidenciais.

 <-label->: Serve para criar rótulos para os campos do formulário. O atributo for faz a associação do rótulo com o campo de entrada correspondente. Isso melhora a acessibilidade e permite que o usuário clique no rótulo para focar no campo de entrada.

 <-input->: Usado para criar campos de entrada de texto. O atributo type define o tipo de campo (por exemplo, texto, número, senha, e-mail):

1. type="text": Campo de texto simples.

2. type="email": Campo específico para e-mails, com validação automática de formato.

3. type="password": Campo de senha, que oculta o texto digitado.

4. type="number": Campo para números.

 <-textarea->: Usada para campos de texto de várias linhas, como caixas de mensagem ou comentários. O conteúdo é escrito por meio de várias linhas.

 <-button->:Cria um botão que envia o formulário. O tipo "submit" indica que o botão é usado para enviar o formulário.
