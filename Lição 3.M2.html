Exercícios

Agora que você já sabe o que é o CSS e como ele funciona, vamos aprender a criar e aplicar CSS no Visual Studio Code (VS Code). Vamos seguir o passo-a-passo para configurar o seu primeiro arquivo CSS.

• Passo 1: Abra o Visual Studio Code.

• Passo 2: Crie uma nova pasta no seu computador para o projeto. Por exemplo, crie uma pasta chamada meu primeiro projeto css.

• Passo 3: No VS Code, clique em File > Open Folder e selecione a pasta meu primeiro projeto css.

• Passo 4: Clique com o botão direito dentro da pasta no painel do VS Code e selecione New File.

• Passo 5: Nomeie o arquivo como index.html.

• Passo 6:Dentro do index.html, coloque o código básico de uma página HTML.

N.B.: Não se esqueca de colocar a tag <-link rel="stylesheet" href="style.css"->, que faz o link entre o HTML e o arquivo CSS.

• Passo 7: Clique com o botão direito novamente e selecione New File.

• Passo 8: Nomeie o arquivo como style.css.

• Passo 9: Agora, dentro do arquivo style.css, você pode adicionar as regras de estilo para o HTML.


Agora que já sabemos como criar um arquivo CSS, vamos aprender como visualizar o resultado no navegador:

• Passo 1: No File Explorer, clique com o botão direito no arquivo index.html e selecione Open With e seleccione o navegador com o qual deseja visualizar o resultado.

• Passo 2: O navegador exibirá sua página com os estilos aplicados pelo CSS.


Exercícios Práticos

Agora que você já sabe como criar os arquivos e aplicar o CSS, vamos praticar um pouco mais. Abaixo estão alguns exercícios para você aplicar o que aprendeu.

Exercício 1:

Crie um arquivo HTML nomeado Site.html e um arquivo CSS nomeado style.css. No arquivo HTML, o conteúdo é à sua escolha.

No seu arquivo style.css, adicione estilos para o seguinte:

• O título <-h1-> deve ser verde.

• O texto do parágrafo <-p-> deve ser vermelho.

• O texto do <-h1-> deve usar a fonte "Arial" e deve ser centralizado.

• O texto do <-p-> deve usar a fonte "Georgia", ser de tamanho 18px e o alinhamento à esquerda.

Exercício 2:

Neste exercício, você terá que criar uma página simples de exemplo e aplicar CSS Interno, CSS Externo e CSS em Linha para estilizar os elementos. Vamos criar um layout básico com um título, parágrafos, uma lista de itens e links. Você usará as três formas de aplicar CSS para entender as diferenças e as situações em que cada uma deve ser utilizada.

Objetivo do Exercício:

• Criar uma página HTML com um título, parágrafo, lista e links.

• Estilizar a página usando:

 CSS Interno: Dentro da tag <-style-> no <-head->.

 CSS Externo: Em um arquivo css separado.

 CSS em Linha: Dentro do próprio HTML, aplicando diretamente aos elementos com o atributo style.
