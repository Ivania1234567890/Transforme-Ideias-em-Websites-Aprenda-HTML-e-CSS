Modelagem de Caixa

O que é o Box Model?

Pensa na caixa de um presente. Para que você tenha um presente embalado, você precisa do conteúdo dentro, depois envolve com uma embalagem (o papel), coloca uma fita (borda) e, por fim, coloca o presente dentro de uma caixa maior, com algum espaço extra para que ele não fique apertado. O box model no CSS é exatamente isso, mas com as caixas de elementos na web.

Cada elemento da página (como um parágrafo, uma imagem) é desenhado dentro de uma caixa. E dentro dessa caixa, existem diferentes partes, como o conteúdo, o preenchimento, a borda e a margem. Vamos conhecer cada uma dessas partes?


Partes do Box Model:

Imagine que temos uma caixa retangular. Ela pode ser dividida em quatro áreas principais:

a) Conteúdo (Content)

O conteúdo é o que está dentro da caixa. Pode ser texto, imagens — qualquer coisa que você coloque dentro do elemento HTML.

Exemplo: O texto dentro de uma <-div->.

Tamanho: O tamanho dessa área é determinado pelas propriedades width e height. Ou seja, a largura e a altura do elemento.

b) Preenchimento (Padding)

Ao redor do conteúdo, temos o preenchimento (padding), que cria um espaço entre o conteúdo e a borda. Imagine que você colocou o presente dentro da caixa e deixou uma distância entre o presente e a embalagem.

Propriedade: A propriedade padding no CSS controla esse espaço. Você pode definir o valor de forma igual para todos os lados ou diferente para cada lado (topo, direita, fundo e esquerda).

Exemplo de CSS para definir o padding:

div {

padding: 20px; (Deixa 20px de espaço ao redor do conteúdo)

}

c) Borda (Border)

A borda (border) é a linha ao redor do conteúdo e do padding, como se fosse a fita que você coloca ao redor da embalagem do presente. Ela pode ser de diferentes tipos: sólida, pontilhada, com cores variadas, e com espessuras diferentes.

Propriedade: A borda é definida com as propriedades border, border-width, border-style e border-color.

Exemplo de CSS para borda:

div {

border: 2px solid black; (Borda de 2px de espessura, linha sólida e cor preta)

}

d) Margem (Margin)

Por fim, temos a margem. A margem cria o espaço entre a borda da caixa e outros elementos ao redor. É o espaço externo da caixa. Imagine que você colocou várias caixas uma ao lado da outra, e a margem é o espaço que separa essas caixas.

Propriedade: A propriedade margin controla esse espaço.

Exemplo de CSS para margem:

div {

margin: 10px; (Cria um espaço de 10px ao redor da borda da caixa)

}


Como tudo isso afeta o layout?

Agora, imagine que você tem um elemento com tamanho fixo, como uma caixa de 300px de largura e 200px de altura. Se você adicionar padding e borda, o tamanho final da caixa vai aumentar, porque o espaço ocupado por ela vai incluir também o padding e a borda.

a) Box Model Padrão (content-box)

No modelo padrão, o tamanho da caixa que você define com as propriedades width e height se aplica apenas ao conteúdo. O padding e a borda vão ser adicionados fora dessa área.

Por exemplo:

div {

width: 300px;

height: 200px;

padding: 20px;

border: 5px solid black;

}

A caixa vai ter 300px de largura e 200px de altura no conteúdo.

No entanto, o tamanho total da caixa vai ser maior do que isso, porque o padding e a borda são adicionados fora dessa área:

Largura total= 300px (conteúdo) + 20px (padding à esquerda) + 20px (padding à direita) + 5px (borda à esquerda) + 5px (borda à direita)= 350px.

Altura total= 200px (conteúdo) + 20px (padding em cima) + 20px (padding embaixo) + 5px (borda em cima) + 5px (borda embaixo) = 250px.

Ou seja, a largura e a altura definidas no CSS representam apenas o conteúdo. Para o layout total, você precisa considerar o padding e a borda também.

b) Box Model border-box

Agora, imagine que você quer definir o tamanho total da caixa, incluindo o padding e a borda, sem que isso aumente o tamanho da caixa.

Nesse caso, usamos o modelo border-box. Quando você usa border-box, a largura e a altura definidas no CSS já incluem o conteúdo, o padding e a borda. Ou seja, a caixa vai ter exatamente o tamanho que você especificou.

Você pode ativar o border-box assim:

* {

box-sizing: border-box;

}

Isso faz com que todos os elementos da página usem o modelo border-box.


O que é posicionamento de elementos?

O posicionamento em CSS é a maneira como você controla a localização dos elementos dentro de uma página web. Quando você posiciona um elemento, você determina onde ele vai estar em relação a outros elementos ou à tela. Existem várias estratégias de posicionamento em CSS, cada uma com seu comportamento específico. Eis abaixo os principais tipos, mas nós iremos abordar somente de dois.


Tipos de Posicionamento em CSS

• position: static

• position: relative

• position: absolute

• position: fixed

Vamos entender cada um deles!

1. Position: static

O static é o padrão para todos os elementos, ou seja, se você não definir um posicionamento específico para um elemento, ele terá o valor static por padrão. Isso significa que o elemento será posicionado de acordo com o fluxo normal do documento.

Como funciona?

Um elemento com position: static é colocado no layout de acordo com o fluxo normal da página, ou seja, ele não será afetado por propriedades como top, right, bottom ou left.

Exemplo:

p {

position: static;

}

Este comportamento é o mesmo que não ter a propriedade position definida.

2. Position: relative

Quando você define um elemento com position: relative, ele ainda vai seguir o fluxo normal do documento, mas você pode deslocá-lo em relação à sua posição original usando as propriedades top, right, bottom ou left.

Como funciona?

O elemento é posicionado de forma relativa ao seu local original no fluxo do documento.

As propriedades top, right, bottom e left deslocam o elemento a partir do seu ponto de origem sem afetar o layout dos outros elementos.

Exemplo:

p {

position: relative;

top: 20px; (Move o elemento 20px para baixo)

left: 30px; (Move o elemento 30px para a direita)

}

Nesse exemplo, o p será deslocado 20px para baixo e 30px para a direita, mas o espaço onde ele estava originalmente no layout ainda será "reservado".
