Introdução Ao CSS

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem usada para definir o estilo de um site. Você pode pensar no CSS como a "roupa" que os elementos de um site vão vestir.

Por exemplo, o HTML é como a estrutura do seu site — ele é como o esqueleto de uma pessoa. Ele diz o que existe na página, como títulos, parágrafos, imagens, listas, etc. Mas o CSS é o que dá beleza e personalidade a esse conteúdo. Ele é responsável por tudo o que tem a ver com cores, fontes, espaçamento e layout.

Então, sempre que você acessar uma página na internet, o que você vê — as cores, as fontes, o espaçamento entre os elementos — são definidos pelo CSS. Exemplo Prático:

Imagina que você tem um arquivo HTML que é algo assim:

                        
<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<title>Minha Página</title>
</head>
<body>
<h1>Bem-vindo ao meu site!</h1>
<p>Este é um parágrafo de exemplo.</p>
</body>
</html>

Agora, você quer que o título (h1) seja azul e o parágrafo (p) tenha um tamanho de fonte maior. Para isso, você escreve o CSS assim:

h1 {

color: blue;

}

p {

font-size: 20px;

}

Como o CSS "Cascata" Funciona?

Agora, o "cascata" (do termo Cascading de CSS) é um conceito importante que significa que, se você tiver várias regras de estilo aplicadas ao mesmo elemento, o navegador vai seguir uma ordem de prioridade para decidir qual estilo aplicar. Ou seja, ele vai escolher o estilo de acordo com o que tem mais especificidade ou, se for igual, a última regra que aparecer no código. Por exemplo, se você escrever isso:

h1 {

color: blue;

}

h1 {

color: red;

}

O título h1 vai ficar vermelho porque a segunda regra foi a última a ser lida, então ela "sobrescreve" a primeira.


Estrutura Básica de uma Regra CSS

Uma regra CSS é composta por três partes principais:

Seletor: Define quais elementos HTML serão estilizados.

Propriedades: São as características ou atributos dos elementos que queremos alterar.

Valores: Especificam o que será aplicado à propriedade.

A estrutura básica é assim:

seletor {

propriedade: valor;

}


Por exemplo:

h1 {

color: blue;

font-size: 24px;

}


Neste exemplo:

h1 é o seletor que seleciona todos os elementos h1 da página.

color e font-size são as propriedades.

blue e 24px são os valores atribuídos a essas propriedades.


A História do CSS

Agora que já entendemos como o CSS funciona, é interessante saber um pouco sobre sua história:

• 1996: O CSS foi criado pelo W3C (World Wide Web Consortium). Antes do CSS, tudo era feito no HTML e não havia uma forma simples de estilizar as páginas.

• 1997-1998: O CSS2 trouxe mais recursos, como a possibilidade de posicionar elementos de maneira mais avançada e de usar fontes diferentes.

• 2000-2010: A evolução do CSS foi marcada pela criação de novas técnicas e propriedades, como o CSS3, que permitiu criar animações, gradientes, sombras e layouts mais modernos e flexíveis.

• 2010 em diante: A partir de 2010, o CSS foi amplamente utilizado para criar layouts responsivos (que se adaptam a diferentes tamanhos de tela, como celulares e desktops). Também surgiram ferramentas como Sass e Less que facilitaram a escrita do CSS, permitindo o uso de recursos como variáveis e funções.
