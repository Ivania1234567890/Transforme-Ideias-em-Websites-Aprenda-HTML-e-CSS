Seletores e Propriedades

Agora, vamos passar para os seletores e propriedades comuns no CSS, dividindo por categorias para facilitar o entendimento.

Seletores:

• Seletores de Elemento (ou Tag): Ele é responsável por selecionar todos os elementos de um tipo específico.

Por exemplo: Seleciona todos os <-p-> na página.

p {

color: blue;

}

• Seletores de Classe: O atributo class é usado para agrupamento de elementos. Um único class pode ser atribuído a múltiplos elementos na página, permitindo aplicar estilos e manipular esses elementos em conjunto.

Exemplo de uso do class:

HTML 
        
<p class="destaque">Este parágrafo tem a classe 'destaque'.</p>
<p class="destaque">Este também tem a classe 'destaque'.</p>
<p class="normal">Este não tem a classe 'destaque'.</p>
CSS

.destaque {

color: red;

font-weight: bold;

}

Resultado no navegador:

Este parágrafo tem a classe 'destaque'.

Este também tem a classe 'destaque'.

Este não tem a classe 'destaque'.

• Seletores de ID: O atributo id serve para identificar de forma única um elemento dentro de uma página HTML. Ou seja, cada valor de id deve ser único em uma página.

Exemplo de uso do id:

HTML 
    
<p id="paragrafo1">Este é um parágrafo com um ID único.</p>
<p id="paragrafo2">Este é outro parágrafo com outro ID.</p>

CSS

#paragrafo1 {

color: blue;

font-weight: bold;

}

Resultado no navegador:

Este é um parágrafo com um ID único.

Este é outro parágrafo com outro ID.

• Seletores Universais: Seleciona todos os elementos na página.

Por exemplo: Aplica um estilo a todos os elementos da página.

{

font-family: Arial, sans-serif;

}

• Seletores de Atributo: Seleciona elementos com um atributo específico.

Por exemplo: Seleciona todos os links (<-a->) que têm o atributo href.

a[href] {

color: green;

}

• Seletores Descendentes: Seleciona elementos dentro de outro elemento.

Por exemplo: Seleciona todos os <-p-> dentro de um <-div->.

div p {

margin-bottom: 10px;

}

• Seletores de Filhos Diretos: Seleciona elementos que são filhos diretos de outro elemento.

Por exemplo: Seleciona todos os <-li-> que são filhos diretos de um <-ul->.

ul: li {

list-style-type: square;

}

• Seletores de Pseudo-classes: Seleciona elementos em um estado específico.

Por exemplo: hover aplica um estilo quando o usuário passa o mouse sobre o elemento.

a: hover {

color: red;

}


Propriedades Comuns no CSS

Aqui estão algumas das propriedades mais comuns no CSS para estilizar elementos na página.

Propriedades de Texto e Fonte

• color: Define a cor do texto.

p {

color: red;

}

• font-size: Define o tamanho da fonte.

h1 {

font-size: 32px;

-

}

• font-family:Define a fonte utilizada.

p {

font-family: Arial, sans-serif;

}

• font-weight:Define o peso da fonte (como negrito).

p {

font-weight: bold;

}

• line-height:Define a altura da linha, ou o espaçamento entre as linhas de texto.

{

p

line-height: 1.6;

}

• text-align: Define o alinhamento do texto.

h1 {

text-align: center;

}

• text-decoration: Adiciona efeitos ao texto, como sublinhado, riscado, etc.

a {

text-decoration: underline;

}

• letter-spacing: Define o espaço entre as letras.

p {

letter-spacing: 2px;

}

• word-spacing: Ajusta o espaço entre as palavras em um texto.

p {

word-spacing: 4px;

}


Esses são alguns dos seletores e propriedades mais usados no CSS.
