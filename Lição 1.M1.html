Estrutura, Tags e Elementos

Estrutura básica do HTML

A estrutura básica de um documento HTML é bem simples, mas é fundamental entender como ela funciona, pois é a partir dessa estrutura que toda a página web é construída. Ela possui a seguinte estrutura:

            
<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Meu Site</title>
</head>
<body>
</body>
</html>

Explicação de cada Parte:

 !DOCTYPE html:

É a declaração que informa ao navegador que este documento é uma página HTML5. Sem essa linha, o navegador pode não renderizar corretamente a página. É a primeira coisa a aparecer em qualquer documento HTML.

 html lang="pt-br":

A tag envolve todo o conteúdo da página; o atributo lang="pt-br" indica que o idioma da página é português (Brasil). Isso ajuda os navegadores e ferramentas de acessibilidade a processar o conteúdo corretamente.

 head:

O head contém informações meta sobre o site (não visíveis diretamente para o usuário), como o título, codificação de caracteres e links para estilos (CSS), etc. Dentro do head, temos:

meta charset="UTF-8":Define a codificação de caracteres, garantindo que o conteúdo seja exibido corretamente, incluindo caracteres especiais, como acentos.

meta name="viewport" content="width=device-width, initial-scale=1.0": Isso ajuda a tornar a página responsiva (adaptável a diferentes tamanhos de tela, como celulares e tablets).


 title:

dentro do head também temos o title que é o título da página, que aparece na aba do navegador. No exemplo, "Meu Site".

 body:

O body contém o conteúdo visível da página — o que o usuário realmente vê ao acessar o site.

O que é uma Tag?

A tag é o marcador usado para identificar um elemento em HTML. Ela indica o início ou o final de um conteúdo ou estrutura dentro do código HTML. As tags de Abertura indicam o início de um elemento. As tags de Fechamento indicam o final de um elemento. As tags são envolvidas por sinais de menor (<) e maior (>).

• Exemplos de Tags:

<-h1->: Tag de abertura do título principal.

<-/h1->: Tag de fechamento do título principal.

<-p->: Tag de abertura para um parágrafo.

<-/p->: Tag de fechamento para um parágrafo.

>As tags de fechamento geralmente são a mesma tag de abertura, mas com uma barra (/) antes do nome da tag.

O que é um Elemento?

Um elemento é a completa estrutura de código que é formada pela tag de abertura, seu conteúdo e a tag de fechamento. Ou seja, o elemento é tudo o que está dentro da tag de abertura e da tag de fechamento. Por exemplo, um elemento HTML inclui:

A tag de abertura: que define onde o conteúdo começa.

O conteúdo: que é o que está sendo descrito ou exibido.

A tag de fechamento: que indica onde o conteúdo termina.

                
<HTML>
<p>Isto é um parágrafo</p>
</HTML>

Juntos, a tag de abertura (<-p->), o conteúdo (Isto é um parágrafo) e a tag de fechamento (<-/p->) formam o elemento <-p->.

• Diferença entre Tags e Elementos:

Tag: Refere-se apenas à parte de marcação, como <-p->, <-h1->, <-img->, etc. As tags são as "etiquetas" que você escreve no código.

Elemento: Refere-se ao conjunto completo, ou seja, a tag de abertura, o conteúdo e a tag de fechamento. Então, um elemento HTML inclui tudo que compõe a estrutura de uma marcação HTML.

N.B.: Algumas tags não precisam de uma tag de fechamento, ou seja, são auto-fechantes. Elas são usadas para inserir conteúdo que não tem "fim" ou "fechamento". Exemplos comuns são as tags de imagem ou linha horizontal.

Aqui estão algumas das tags HTML mais usadas e uma explicação simples de como cada uma funciona. Essas tags são fundamentais para estruturar o conteúdo em uma página web.

1. <-h1->, <-h2->,... <-h6->

Essas são as tags de título. O <-h1-> representa o título principal da página. Em uma página, o <-h1-> geralmente deve ser o título mais importante, e as tags <-h2-> até <-h6-> são usadas para criar subtítulos.

2. <-p->

O <-p-> define um parágrafo de texto. Ele organiza o texto na página.

3. <-a->

O <-a-> cria um link. O atributo href indica para onde o link vai.

Por exemplo, <-a href=https://www.google.com target=”_blank”>Clique aqui para ir ao Google<-/a->.

No exemplo, o link leva para o Google. Quando você clica no texto "Clique aqui para ir ao Google", ele te leva a esse endereço. O atributo blank abre a hiperligação numa nova janela ou separador.

Para além destas, existem mais 3 propriedades diferentes:

 Self: A página de destino abrirá na mesma janela ou guia.

 Parent: Abre a hiperligação numa frame com ligação à actual.

 Top: A página será aberta na janela inteira (sem frames), substituindo qualquer estrutura de frames.

Também tem a hiperligação local, que é um link que aponta para um arquivo ou recurso que está dentro do mesmo diretório ou pasta do seu projeto. Por exemplo: <-a href="sobre.html">Ir para a página Sobre<-/a->

Imagine que você tem duas paginas no seu directório. Uma chamada índex.html e outra sobre.hmtl. A página sobre.html está no mesmo diretório que index.html, então usamos apenas o nome do arquivo sobre.html como o caminho. Quando o usuário clicar no link, o navegador irá carregar a página sobre.html (que está na mesma pasta).

4. <-img->

A tag <-img-> é usada para inserir imagens em uma página. A tag <-img-> requer um atributo src que indica o caminho da imagem, um atributo alt que fornece uma descrição alternativa da imagem (importante para acessibilidade) e também o atributo width que especifica, em pixels, o tamanho da imagem ao ser exibida na página.

Por exemplo, <-img src="imagem.jpg" width=”100px “ alt="Descrição da imagem"->.

Além de usar texto para criar links, você pode usar imagens dentro de links. Para isso, você coloca a tag <-img-> dentro de uma tag <-a->, transformando a imagem em um link clicável.

Exemplo de uma imagem como link:

<-a href="https://www.exemplo.com"><-img src= "imagem.jpg" width="100px" alt="Descrição da imagem"><-/a->

5. <-br->

A tag <-br-> insere uma quebra de linha.

Por exemplo, <-p->Este é um parágrafo de texto.<-br->Agora começa uma nova linha.<-/p->

Quando você usa <-br->, o conteúdo que vem depois dele é exibido na linha seguinte. É muito útil para quando você não quer usar um novo parágrafo, mas deseja apenas separar o conteúdo.

6. <-hr->

A tag <-hr-> é usada para inserir uma linha horizontal na página. Essa linha é frequentemente usada para separar seções de conteúdo ou criar uma divisão visual entre diferentes partes da página. A tag <-hr-> é uma tag auto-fechante, isso significa que ela não precisa de uma tag de fechamento. Basta escrever <-hr->, e a linha horizontal será inserida.

7. <-b->

A tag <-b-> é usada para negrito. Ela serve para dar ênfase visual ao texto, deixando-o mais escuro e destaque no conteúdo. Veja o exemplo:

<-p->Este é um texto normal, mas <-b->este está em negrito<-/b->.<-/p->

Resultado: Este é um texto normal, mas este está em negrito.

8. <-i->

A tag <-i-> é usada para itálico. Ela altera o estilo do texto para incliná-lo (em comparação ao texto normal). Veja o exemplo:

<-p->Este é um texto normal, mas <-i->este está em itálico<-/i->.<-/p->

Resultado: Este é um texto normal, mas este está em itálico.

9. <-small->

A tag <-small-> é usada para reduzir o tamanho do texto. O objetivo dela é diminuir a tamanho da fonte do texto em relação ao texto ao seu redor. O comportamento visual da tag é, portanto, o de fazer o texto aparecer menor, mas não muda o significado do conteúdo de maneira semântica. Veja o exemplo:

<-p->Este é um texto normal, e <-small->este está em um tamanho menor<-/small->.<-/p->

Resultado: Este é um texto normal, e este está em um tamanho menor.

10. <-mark->

A tag <-mark-> é usada para destacar ou marcar um texto, geralmente com cor de fundo amarela (como se fosse um marcador de texto). Isso é útil quando você quer chamar atenção para algo em um conteúdo, como uma palavra-chave, um termo de pesquisa ou uma parte importante do texto. Veja o exemplo:

<-p->O <-mark->HTML<-/mark-> é uma linguagem de marcação usada para estruturar páginas web.<-/p->

Resultado: HTML é uma linguagem de marcação usada para estruturar páginas web.

11. <-sub->

A tag <-sub-> é usada para subscrição, ou seja, para colocar o texto um pouco abaixo da linha de base do texto normal. Isso é comum em fórmulas químicas (por exemplo, H₂O) ou em notações matemáticas. Veja o exemplo:

<-p->A fórmula da água é H<-sub->2<-/sub->O.<-/p->

Resultado: A fórmula da água é H2O.

12. <-sup->

A tag <-sup-> é usada para superscrição, ou seja, para posicionar o texto um pouco acima da linha de base do texto normal. Isso é muito comum em potências matemáticas (como 2²) ou em notações de pé de página. Veja o exemplo:

<-p->A fórmula para o quadrado de um número é x<-sup->2<-/sup->.<-/p->

Resultado: A fórmula para o quadrado de um número é x2.

Então, estas são algumas tags amplamente usadas em construção de páginas web.
