Introdução Ao HTML

O que é HTML?

O HTML (HyperText Markup Language - Linguagem de Marcação de Hipertexto) é a linguagem de marcação que usamos para criar páginas na web. Imagine que você está montando uma página de um site. O HTML vai ser como o "esqueleto" da página, ou seja, ele vai definir a estrutura do conteúdo, como:

• Títulos

• Parágrafos

• Imagens

• Links

• Listas

Então, o HTML serve para organizar tudo o que você vê numa página de internet.

Qual é a história do HTML?

1. Início (1989-1991):

Em 1989, Tim Berners-Lee, um cientista, estava criando uma maneira de conectar documentos na web. Ele criou o HTML, uma linguagem simples, para ajudar a organizar essas informações. Em 1991, ele lançou o primeiro site da web usando HTML.

2. Primeiras Melhorias (1995-1997):

Nos anos seguintes, o HTML começou a evoluir e ganhar novas funcionalidades, como a possibilidade de criar tabelas, links e formulários, o que melhorava a aparência e a interatividade das páginas.

3. HTML 4.0 (1997):

Em 1997, o HTML recebeu uma grande atualização, chamada de HTML 4.0, que começou a separar melhor o conteúdo da apresentação. Ou seja, o HTML começou a se preocupar mais com como estruturar o conteúdo, enquanto o CSS (Cascading Style Sheets) passou a ser usado para definir o estilo da página (cores, fontes, etc.).

4. HTML5 (2014 - Atualidade):

Em 2014, foi lançada a versão HTML5, que trouxe muitas melhorias! Ele permitiu, por exemplo:

 Criar elementos de estrutura mais claros (como header, footer).

 Tornar os sites mais interativos e compatíveis com celulares e tablets.
