CSS Interno, Externo e Em Linha

Agora, vamos aprender sobre as três formas principais de aplicar CSS em uma página web: CSS Interno, CSS Externo e CSS em Linha. Cada um desses métodos tem suas características e é mais adequado para situações específicas. Então, vamos detalhar cada um deles.

I. O que é CSS Interno?

O CSS Interno é quando você coloca as regras de estilo dentro da própria página HTML, dentro da tag <-style->, que fica normalmente no head da página. Ou seja, o CSS fica "embutido" no próprio código HTML.

Como usar?

O CSS Interno é definido dentro da tag <-style-> que deve ser colocada dentro da seção <-head-> do HTML. Veja um exemplo:

        
<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Exemplo de CSS Interno</title>
<style>

body {

font-family: Arial;

background-color: #f0f0f0;

}

h1 {

color: blue;

text-align: center;

}

p {

color: white;

font-size: 16px;

}


</style>
</head>
<body>
<h1>Bem-vindo ao CSS Interno!</h1>
<p>Este é um exemplo de como usar CSS diretamente dentro do HTML.</p>
</body>
</html>

Quando usar o CSS Interno?

• Quando você precisa de estilos específicos para uma única página e não quer criar um arquivo CSS separado para isso.

• Projetos pequenos ou quando você está fazendo um protótipo e quer testar rapidamente os estilos na própria página.

• Se os estilos não serão reutilizados em outras páginas, o CSS Interno é uma escolha viável.

II. O que é CSS Externo?

O CSS Externo é quando você coloca as regras de estilo em um arquivo CSS separado e, em seguida, o linka a esse arquivo dentro do HTML. Ou seja, você cria um arquivo .css onde ficará todo o código de estilo, e o HTML se refere a esse arquivo para aplicar os estilos.

Como usar?

Você cria um arquivo separado com a extensão .css e depois faz um link para esse arquivo dentro da tag <-head-> do HTML usando a tag <-link->. Veja um exemplo:


<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Exemplo de CSS Externo</title>
<link rel="stylesheet" href="styles.css"> -( Link para o arquivo CSS)
</head>
<body>
<h1>Bem-vindo ao CSS Externo!</h1>
<p>Este exemplo está usando um arquivo CSS separado para os estilos.</p>
</body>
</html>

CSS (styles.css):

body {

font-family: Arial, sans-serif;

background-color: #f0f0f0;

}

h1 {

color: blue;

text-align: center;

}

p {

color: gray;

font-size: 16px;

}

Quando usar o CSS externo?

• Para projetos maiores e quando você tem várias páginas, o CSS Externo é a melhor opção, pois o arquivo de estilos pode ser reutilizado em todas as páginas. Basta colocar o link em todas as páginas.

• Quando você precisa manter o HTML mais limpo e separado do CSS.

• Se você quer aplicar estilos consistentes em várias páginas de um site, o CSS Externo ajuda a centralizar os estilos em um único lugar.

III. O que é CSS em Linha?

O CSS em Linha é quando você aplica os estilos diretamente no próprio elemento HTML, usando o atributo style. Esse estilo é aplicado apenas ao elemento específico em que ele está definido.

Como usar?

Você usa o atributo style dentro de uma tag HTML para definir o estilo diretamente naquele elemento. Veja um exemplo: 

    
<!DOCTYPE html>
<html lang="pt-br">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Exemplo de CSS em Linha</title>
</head>
<body>
<h1 style="color: blue; text-align: center;">Bem-vindo ao CSS em Linha!</h1>
<p style="color: gray; font-size: 16px;">Este é um exemplo de como aplicar estilos diretamente nos elementos HTML.</p>
</body>
</html>

Quando usar o CSS em Linha?

• Para aplicar um estilo único em um único elemento que não será reutilizado.

• Quando você precisa fazer alterações rápidas ou testar algo pontual e não quer modificar o arquivo CSS inteiro.
