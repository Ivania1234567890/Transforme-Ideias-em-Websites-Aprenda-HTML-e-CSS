Exercicios práticos

Antes de realizarmos os exercícios práticos, vamos mostrar primeiro o passo-a-passo para criar um documento HTML no seu dispositivo (usando o aplicativo Visual Studio Code).

• Passo 1: Baixe e instale o VS Code no seu dispositivo através do site oficial do Visual Studio Code;

• Passo 2: No seu computador, crie uma pasta para o seu projecto. Pode ser chamada por exemplo, Meu primeiro site;

• Passo 3: Abra o VS Code. No menu superior, clique em File > Open Folder... (ou Arquivo > Abrir Pasta...). Navegue até a pasta que você criou (Meu primeiro site) e clique em Abrir.

• Passo 4: Dentro da pasta do projeto, clique com o botão direito e escolha New File (ou Novo Arquivo). Nomeie o arquivo como index.html. O arquivo HTML padrão geralmente é chamado de index.html, pois é o primeiro carregado em um site.

Agora que já sabemos como usar o Visual Code, vamos dar inicio aos exercícios.


Exercício 1: Criando uma Página Simples

Objetivo: Criar uma página HTML simples com título, parágrafo, lista e imagem.

Instruções:

 Crie um arquivo HTML com a estrutura básica.

 No corpo da página, adicione:

• Um título principal (<-h1->) com o texto "Minha Página Pessoal".

• Um parágrafo (<-p->) explicando algo sobre você.

• Uma lista não ordenada (<-ul->) com 3 hobbies (use a tag <-li-> para cada item).

• Qualquer imagem (<-img->) que você queira (não se esqueça de usar o atributo src, width e alt).

Exercício 2: Criando um Formulário de Contato

Objetivo: Criar um formulário simples para coletar informações do usuário.

 Crie um formulário com os seguintes campos:

• Nome (campo de texto)

• E-mail (campo de e-mail)

• Mensagem (campo de texto)

• Adicione um botão para enviar o formulário.

• Defina um action fictício, como "enviar.php", para onde os dados seriam enviados (não é necessário que o formulário realmente funcione, apenas a estrutura).
