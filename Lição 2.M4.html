Projeto Final: Criação de um Website Publicitário

Com base em todos os conceitos aprendidos nos módulos anteriores, você irá criar um site publicitário completo. Este projeto final será a oportunidade perfeita para demonstrar o que aprendeu, unindo HTML, CSS e boas práticas de desenvolvimento.


Descrição do Projeto

Desenvolver um website publicitário de uma página para promover um produto ou serviço fictício ou real. Você pode escolher algo de seu interesse, como:

• Um novo smartphone.

• Um restaurante.

• Um evento.

• Um curso.

Requisitos do Projeto

1. Estrutura HTML

O site deve conter:

Cabeçalho (header):

 Título com o nome do produto ou serviço.

Corpo (body):

 Parágrafo detalhando o produto/serviço e por que ele é único.

 Pelo menos 3 imagens relevantes.

 Lista ordenada destacando os benefícios ou características do produto.

 Tabela de preços.

 Informações de contato ou links para redes sociais.

Formulário de contato:

 Deve incluir campos de entrada (<-input->), uma área de texto (<-textarea->), e um botão de envio (<-button->).


2. Estilo CSS

Estilo básico:

o Usar CSS externo para estilizar o site.

Paleta de cores:

o Escolha uma paleta harmoniosa para o site.

Tipografia:

o Use pelo menos 2 fontes diferentes.

o Diferencie estilos entre títulos, parágrafos e links.

Box Model:

o Utilize margin, padding, e border para ajustar espaçamentos e organizar os elementos visualmente.

Transições:

o Adicione transições suaves para imagens e links ao passar o mouse.


3. Regras Responsivas

• O site deve ser responsivo, ou seja, deve se ajustar bem em telas de diferentes tamanhos.
